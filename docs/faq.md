# General FAQ

**Q: What is the default password?**  
A: `admin` / `admin`

**Q: How to factory reset?**  
A: Press and hold reset button for 10+ seconds.

**Q: Device not getting IP address?**  
A: Check cable, PoE switch, or set static IP.

**Q: SIP registration failed?**  
A: Check SIP server address, credentials, and firewall.

**Q: How to update firmware?**  
A: Download latest firmware from Fanvil official site and upload via web interface.

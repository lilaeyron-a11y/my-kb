Fanvil’s flagship enterprise IP phone series (X210, X310, X510, X710, X2120, etc.)

### Key Features
- Up to 20 SIP lines
- HD Audio (G.722, Opus)
- Gigabit Ethernet + PoE
- Color display + DSS keys
- Bluetooth (selected models)

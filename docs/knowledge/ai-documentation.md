# AI and Documentation

AI changes documentation work, but it does not remove the need for documentation strategy.

AI can help draft, summarize, translate, classify, and search content. But the human writer still needs to define the audience, structure, quality bar, source of truth, and maintenance process.

## Where AI can help

- draft first versions
- summarize technical material
- generate examples
- compare documentation structures
- find unclear wording
- create content templates
- support multilingual workflows

## Where human judgment matters

- product accuracy
- information architecture
- user research
- legal and safety risk
- content governance
- voice and tone
- prioritization

## My career hypothesis

The future technical writer will not only write documents. They will design knowledge systems where humans and AI can both find, maintain, and reuse accurate information.

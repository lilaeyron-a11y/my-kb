# Career Notes

I am building this site because I want to move toward a stronger technical communication career.

I am using this transition period to learn, build, and reposition myself for better opportunities.

The goal is not only to find the next job. The goal is to build proof that I can think and work like a documentation professional.

- content strategist

In the long term, I want this portfolio to help me reach:

- better companies

- higher salary opportunities
- consulting opportunities

## What this site should become

This site should become:

- a writing sample
- a proof of docs-as-code ability
- a place to practice documentation strategy
